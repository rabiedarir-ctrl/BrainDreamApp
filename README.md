# BrainDreamApp

تطبيق متكامل للهواتف الذكية يلتقط إشارات الدماغ باستخدام جهاز EEG خارجي، ويحولها إلى قارئ صوتي وصوري، مع مسجل أحلام لتسجيل النشاط أثناء النوم.

## المميزات

- لقط الترددات الدماغية: دلتا، ثيتا، ألفا، بيتا، جاما
- قارئ صوتي: يحوّل الحالة العقلية أو مرحلة النوم إلى صوت
- قارئ صوري: يولد صورة AI تمثّل الحالة العقلية
- مسجل أحلام: تسجيل EEG أثناء النوم، مع تقرير صباحي

## المتطلبات

- جهاز EEG خارجي (Emotiv، NeuroSky...)
- هاتف Android يدعم Bluetooth
- مكتبة BrainFlow
- Android Studio 2023 أو أحدث

## كيفية التشغيل

1. افتح المشروع في Android Studio
2. تأكد من إضافة مكتبات BrainFlow و Text-to-Speech
3. شغّل التطبيق على الهاتف مع توصيل جهاز EEG
4. اتبع التعليمات على الشاشة لبدء لقط الترددات أو تسجيل النوم
